# 宿舍管理系统 java1124

## 项目概述

宿舍管理系统是一个基于Java + SpringBoot + Vue实现的前后端分离项目。该项目旨在提供一套完整的代码、数据库及文档，以方便管理宿舍的日常事务。

## 技术栈

- 架构：B/S、MVC
- 系统环境：Windows
- 开发环境：
  - IDEA
  - JDK 17
  - Mysql
- 技术栈：
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

- **管理员登录**：支持验证码功能
  - 登录验证码登录
- **学生管理**：
  - 增删改查功能
- **楼层管理**：
  - 增删改查功能
- **宿舍情况管理**：
  - 增删改查功能

## 系统角色

- 管理员：负责系统的登录、学生信息、楼层信息、宿舍情况的管理

## 运行环境

- 操作系统：Windows
- 数据库：Mysql
- 服务器：需支持SpringBoot应用的运行
